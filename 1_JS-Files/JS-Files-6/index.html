Інструкція if(...)

Дана інструкція виконує блок коду, якщо умова в круглих дужках дає результат true. 

                
let message = "Hello World!"

if (2 > 1) {
    console.log (message);
}

У якості операндів, всередині умови також можна використовувати змнінні. Також умова може бути складною: містити різні логічні оператори.

Після обчислення виразу в круглих дужках інструкція if(...) перетворює результат в логічний тип.

Інструкція if(...) має скорочену форму запису для випадків, коли необхідно виконати якусь одну інструкцію. 

                
if (2 > 1) console.log ("Hello World!");

Інструкція if(...) може мати необов'язковий блок else (інакше) - він виконується, коли умова неістинна. 

                
if ( 2 < 1) {
    alert ("Умова істинна.")
} else {
    alert ("Умова неістинна."); // спрацює цей блок
}

У випадках, коли потрібно перевірити декілька варіантів умов, використовується блок else if (...), але можна перевярти декілька умов без використання else. 

                
let year = prompt ("Коли була опублікована специфікація ECMAScript-2015?", "");

if (year < 2015) {
    alert ("Зарано.");
} else if (year > 2015) {
    alert ("Запізно.");
} alert ("Вірно!");

Умовний (тернарний) оператор "?"

Часто бувають випадки, коли необхідно визначити змінну залежно від умови: 

                
let accessAllowed;

let age = prompt ("Скільки Вам років?", "");

if (age > 18) {
    accessAllowed = true;
} else {
    accessAllowed = false;
}

alert (accessAllowed); // true/false, залежно від уведеного користувачем значення

Використовуючи умовний (тернарний) оператор "?" можна виконати вищенаведену операцію значно коротшим чином: 

                
let age = prompt ("Скільки Вам років?", "");

let accessAllowed = (age > 18) ? true : false;

alert (accessAllowed); // true/false, залежно від уведеного користувачем значення

Спочатку обчислюється умова: якщо вона істинна, то повертається перше значення, а у протилежному випадку повертається друге значення.

Також, у прикладі вище дужки можна забрати, оскільки умовний (тернарний) оператор "?" має низький приорітет.

Умовний оператор "?" називають ще тернарним, бо він приймає 3 операнди і є єдиним у своєму роді.

Цей оператор також можна використовувати для запису кількох умов у вигляді послідовності операторів "?". 

                
let age = prompt ("Вік?", "18");

let message = (age < 3) ? "Привіт, малюк!" :
    (age < 18) ? "Привіт!" :
    (age < 100) ? "Доброго дня!" :
    "Який незвичайний вік!"

alert (message);

Така реалізація може бути не зручною для читання, тому рекомендується використовувати класичну реалізацію через інструкцію if (...).

Існує нетрадиційний спосіб використання оператора "?", а саме його можна використовувати для заміни інструкції if (...). 

                
let company = prompt ("Яка компанія створила JS?", "");

(company == "Netscape") ?
    alert ("Вірно!") : alert ("Невірно!"); // Вірно!

В даному вище прикладі, залежно від результату обчислення умови, буде виконана перша частина (якщо true), або друга частина (якщо false).

Проте не рекомендується використовувати оператори "?" таким чином, оскільки це погіршує читання коду. Оператор "?" варто використовувати лише тоді, коли потрібно повернути одне чи друге значення залежно від умови.
